# AI-FED Landing Page

Next.js wrapper for deploying `landing_ver4.html` to Vercel.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Keep the default framework preset as `Next.js`.
4. Deploy.

The root route `/` serves `public/landing_ver4.html`.
